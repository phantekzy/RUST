## Learning Roadmap

This repository follows a structured reading list to master Rust, moving from core concepts to low-level systems programming.

### 1. The Rust Programming Language (Current)
* **Status:** In Progress
* **Focus:** Syntax, ownership, borrowing, lifetimes, and the standard library. This is the official baseline guide for understanding the language rules.

### 2. Programming Rust (Next)
* **Focus:** Deep dive into memory layout, compiler behavior, and language internals. This book explains *why* Rust operates the way it does under the hood and covers advanced traits, async, and FFI.

### 3. Rust in Action
* **Focus:** Systems programming mechanics. It provides practical projects like building file systems, CPU emulators, and raw network packet handlers to see how Rust interacts with hardware and system resources.

### 4. Rust for Rustaceans
* **Focus:** Intermediate and advanced idiomatic patterns. Written for engineers who know the basics but want to break through design plateaus, focusing on library architecture, unsafe code guidelines, and macro design.

### 5. Rust Atomics and Locks
* **Focus:** Low-level concurrency and synchronization. This focuses on the exact physics of memory ordering, building custom threading primitives, and managing atomic operations safely.
